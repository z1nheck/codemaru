# Codemaru Privacy Policy

**Effective date:** 2026-06-18
**Contact:** zandre.tkw@gmail.com

Codemaru ("the extension") is a browser extension that translates the text of the
web page you are currently viewing into your chosen language, using Chrome's
**built-in, on-device AI** (the Translator / Prompt API powered by Gemini Nano).

## Summary

Codemaru does **not** collect, store, transmit, share, or sell any user data.
All translation happens locally on your device. No data ever leaves your computer,
and there are no external servers, analytics, trackers, or advertising.

## What the extension accesses, and why

- **Web page content (the text of the current tab):** When you click the Codemaru
  toolbar button, the extension reads the visible text of the active page so it can
  be translated. This text is passed only to Chrome's on-device AI model and is used
  solely to produce the translation shown back on the same page. It is processed in
  memory and is never saved or sent anywhere.

That is the only data the extension touches.

## Data collection and sharing

- We do **not** collect or retain any personal or sensitive information.
- We do **not** transmit any data off your device.
- We do **not** share or sell any data to third parties.
- We do **not** use any analytics, advertising, or tracking.
- We do **not** use the data for any purpose other than performing the translation
  you requested.

## Permissions

- **activeTab** — lets the extension read the current tab's text, only after you
  click the toolbar button, so it can be translated.
- **scripting** — lets the extension run its translation script on the current tab
  and insert the translated text back into the page.

Codemaru requests no broad host permissions and runs only on the tab you explicitly
act on.

## Limited Use disclosure

The use of information received through Chrome APIs by Codemaru adheres to the
Chrome Web Store User Data Policy, including the Limited Use requirements. Data
accessed by the extension is used solely to provide the user-facing translation
feature and is never transferred, sold, or used for any other purpose.
